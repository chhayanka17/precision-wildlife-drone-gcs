# 🚁 Precision Wildlife Contraceptive Delivery Drone System

> A Mission Planner–inspired Ground Control Station prototype for humane wildlife population management using autonomous drones.

🏆 **Techdivathon (20–21 Feb) | Top 10 Finish** | Built in 24 hours

---

## 🚀 Key Highlights

- Mission Planner–inspired Ground Control System
- Real-time telemetry simulation
- Wildlife detection workflow (wild boar focus)
- Automated intervention authorization logic
- Edible contraceptive pod deployment concept
- Fail-safe landing & safety simulation

---

## 🌳 Problem Statement

Wild boar overpopulation causes:
- 🌾 Crop destruction and economic losses
- 🌿 Ecosystem imbalance
- ⚠️ Increased human–wildlife conflict

Current solutions like culling or relocation are often controversial, costly, and unsustainable.

---

## 💡 Proposed Solution

| Feature | Description |
|---|---|
| 🗺️ Mission Planning | Interactive satellite map with waypoint configuration |
| 📡 Telemetry Simulation | Real-time simulated drone status and positioning |
| 🎯 Wildlife Detection | Automated wild boar detection workflow |
| ⚙️ Decision Logic | Rule-based intervention authorization system |
| 📦 Pod Deployment | Edible contraceptive pod delivery simulation |
| 📊 Event Logging | Structured mission log and audit trail |
| 🛑 Fail-Safe Systems | Emergency landing and safety recovery simulation |

---

## 🧠 System Workflow
```
Mission Planning → Telemetry Simulation → Wildlife Detection
→ Intervention Decision Logic → Payload Deployment
→ Logging & Safety Monitoring
```
